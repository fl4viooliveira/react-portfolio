  ## Getting Started
  
  ### Prerequisites 

  You must have node.js installed on your computer to install this application.

  

  
  ### Installation 

  1. Clone the repo
  ``` sh
  git clone https://github.com/fl4viooliveira/react-portfolio.git

  cd react-portfolio
  ```

 2. Install NPM packages
 ``` sh
 npm install
 ```

 3. Execute `React`
 ``` sh
 npm start
 ```
